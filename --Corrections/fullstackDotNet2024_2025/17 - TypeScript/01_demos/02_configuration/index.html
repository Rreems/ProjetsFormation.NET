Le fichier tsconfig.json

TypeScript vérifie si un fichier tsconfig existe et utilise les paramètres qui y sont définis.

Avantage : plus besoin de s'inquieter des problèmes de compatibilité entre les versions

  • Le créer manuellement à la racine du projet
  • Utilisez la commande : tsc --init
  • Utiliser un générateur en ligne, par exemple tsconfig.json
  • Une fois le fichier tsconfig.json en place, vous pouvez utiliser la commande : tsc -w
