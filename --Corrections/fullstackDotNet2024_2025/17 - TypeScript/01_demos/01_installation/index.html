Installation de TypeScript

  • Installer node.js
  • Vérifier l'installation de Node.js avec la commande : node -v
  • Installer typescript en global avec la commande : npm install -g typescript
  • Vérifier l'installation de ts tsc -v

Mon premier script TS

  • Ajouter une balise script dans le html avec l'extension .js
  • Créer un fichier .ts, puis dans la console exécuter la commande : tsc nom-fichier.ts
  • Avec l'option -w ou --watch, vous pouvez préciser un fichier à surveiller tsc -w nom-fichier.ts
  • ctrl + c dans le terminal pour arrêter la surveillance
